Installation
####################

netscud requires:

* python 3.7+

and the following python modules:

* async_ssh 2.4.2+
* pyyaml 5.3.1+

Installation is straightforward:

* with pip:

.. raw:: html

   <div style="background-color:rgb(238,255,204);border:1px solid #D9D9D9;padding:12px;">pip install netscud</div><BR><BR>
