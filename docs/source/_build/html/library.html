class netscud.base_connection.NetworkDevice(**kwargs)¶

Base class for network object

Parameters

  • ip (str) – IP address of a device

  • username (str) – Username used to connect to a device

  • password (str) – Password used to connect to a device

  • device_type (str) – Type of device used

  • port (int, optional) – TCP port used to connect a device. Default value is “22” for SSH

  • timeout (int, optional) – TCP port used to connect a device. Default value is 10 seconds

  • _protocol (str, optional) – Protocol used to connect a device. “ssh” or “telnet” are possible options. Default value is “ssh”

  • enable_mode (bool, optional) – Enable mode for devices requiring it. Default value is “False”

  • enable_password (str, optional) – Enable password used for enable mode.

  • conn (SSHClientConnection object) – Variable used for the management of the SSH connection

  • _writer (StreamWriter object) – Variable used for the management of the Telnet connection and writing channel

  • _reader (StreamReader object) – Variable used for the management of the Telnet reading channel

  • possible_prompts (list) – Used by the connect method to list all possible prompts of the device

  • _connect_first_ending_prompt (list) – Default possible ending prompts. Used only the time after login and password to discover the prompt

  • list_of_possible_ending_prompts (list) – Different strings at the end of a prompt the device can get. Used for detecting the prompt returned in sent commands

  • _telnet_connect_login (str) – Login prompt for Telnet. Used to detect when a login is expected or when login and password access is failed

  • _telnet_connect_password (list) – Password prompt for Telnet. Used to detect when a login is expected or when login and password access is failed

  • _telnet_connect_authentication_fail_prompt (list) – Known failing messages or prompts when an authentication has failed. Used to get an answer faster than timeout events

  • cmd_enable (str) – Enable command for entering into enable mode

  • cmd_disable_paging (str) – Command used to disable paging on a device. That command is run at connection time

  • cmd_enter_config_mode (str) – Command used to enter into a configuration mode on a device when this device support that feature.

  • cmd_exit_config_mode (str) – Command used to leave a configuration mode on a device when this device support that feature.

  • cmd_get_version (str) – API command used to get the software version of a device

  • cmd_get_hostname (str) – API command used to get the hostname of a device

  • cmd_get_model (str) – API command used to get the model of a device

  • cmd_get_serial_number (str) – API command used to get the serial number of a device

  • cmd_get_config (str) – API command used to get the running configuration of a device

  • cmd_save_config (str) – API command used to save the running configuration on the device

async telnet_send_command_with_unexpected_pattern(cmd, pattern, error_pattern=None)¶

Async method used to send command for Telnet connection to a device with possible unexpected patterns

send_command can wait till time out if login and password are wrong. This method speed up the returned error message when authentication failed is identified. This method is limited to authentication whem password is required

Parameters

  • cmd (str) – command to send

  • pattern (str) – optional, a list of patterns located at the very end of the a returned string. Can be used to define a custom or unexpected prompt a the end of a string

  • error_pattern (str) – optional, a list of failed prompts found when the login and password are not correct

Returns

the output of command

Return type

str
